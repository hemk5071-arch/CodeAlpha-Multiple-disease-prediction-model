Multiple Disease Prediction System

Overview

The Multiple Disease Prediction System is a machine learning-based healthcare application that predicts the likelihood of various diseases using patient medical data. The system integrates multiple trained machine learning models into a single user-friendly platform, allowing users to select a disease and receive a prediction based on the provided clinical parameters.

This project aims to demonstrate the practical application of Machine Learning in healthcare by combining multiple disease prediction models into one centralized system.

---

Features

- Predicts multiple diseases using Machine Learning models.
- User-friendly web interface built with Streamlit.
- Fast and accurate predictions.
- Interactive input forms for patient data.
- Single platform for multiple disease predictions.
- Easy deployment and scalability.
- Real-time prediction results.

---

Diseases Supported

1. Diabetes Prediction

Predicts whether a patient is likely to have diabetes based on medical attributes such as:

- Number of Pregnancies
- Glucose Level
- Blood Pressure
- Skin Thickness
- Insulin Level
- BMI
- Diabetes Pedigree Function
- Age

2. Heart Disease Prediction

Predicts the likelihood of heart disease using clinical information such as:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol Level
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise-Induced Angina
- ST Depression
- Slope of Peak Exercise ST Segment
- Number of Major Vessels
- Thalassemia

3. Breast Cancer Prediction

Predicts whether a tumor is likely to be malignant or benign based on cell nucleus characteristics such as:

- Mean Radius
- Mean Texture
- Mean Perimeter
- Mean Area
- Mean Smoothness
- Mean Compactness
- Mean Concavity
- Mean Concave Points
- Mean Symmetry
- Mean Fractal Dimension

and several other diagnostic features.

Machine Learning Workflow

Data Collection

Datasets were collected from publicly available healthcare datasets.

Data Preprocessing

- Handling missing values
- Feature selection
- Data cleaning
- Data transformation
- Train-Test splitting

Model Training

Different machine learning algorithms were trained and evaluated on their respective datasets.

Model Saving

Trained models were saved using Pickle:

pickle.dump(model, open('model.pkl', 'wb'))

Model Integration

All trained models were loaded into a single Streamlit application:

import pickle

diabetes_model = pickle.load(open('diabetes_model.pkl', 'rb'))
heart_model = pickle.load(open('heart_model.pkl', 'rb'))
breast_cancer_model = pickle.load(open('breast_cancer_model.pkl', 'rb'))
